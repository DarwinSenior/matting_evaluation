What is Image Matting?

Matting refers to the problem of precise extraction of foreground object in images. It is one of the key techniques in many image editing and film production applications.

There are two inputs for a matting algorithm:
Input 1: Image
Input 2: Trimap

The first input is a RGB image. The second input is called Trimap. A trimap contains three regions: Known Foreground, Known Background and Unknown regions.
  • Known Foreground: the white areas in the trimap. Pixels in this areas belong to the foreground.
  • Known Background: the black areas in the trimap. Pixels in this areas belong to the background.
  • Unknown: the gray areas in the trimap. It is usually the area where very fine details exist, such as hairs and furs. This is also the ONLY area which a matting algorithm needs to solve for.

Given a pair of composition results produced by two different matting methods, user can follow these instructions to judge which method is better:

1. User should ONLY compare the results in the Unknown regions, since results in the Known Foreground and Known Background regions are exactly the same.

2. A bad matting method will produce smear and chunky artifacts, such as the result of method 2. This is mainly caused by a wrong extraction of background as foreground.

3. Pay attention to small and fine details (e.g., hairs and furs) in the Unknown regions.

4. Sometimes a different background image will show or hide details and mistakes. For example, if the background image is black, the smear in the above right image will disappear. So user should compare results on multiple background images.
